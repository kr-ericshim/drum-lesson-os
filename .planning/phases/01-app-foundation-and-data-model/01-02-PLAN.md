<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Student data is globally readable through Supabase API | HIGH | Enable RLS on every Phase 1 domain table and add authenticated instructor ownership policies. |
| Seed data creates permanent dependency on a real auth user | MEDIUM | Use a clearly documented demo instructor identity strategy and keep seed reset instructions explicit. |
| Over-structured enums block real drum lesson notes | MEDIUM | Use enums only for dashboard-critical statuses and categories; keep notes/details text fields. |
</threat_model>
