## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Dashboard leaks all students across instructors | HIGH | Use the instructor-scoped query helper and RLS-backed schema from Plan 01-02. |
| Missing Supabase env creates blank UI | MEDIUM | Render setup/error state with copy from UI-SPEC instead of throwing opaque errors. |
| Dashboard preview uses fake hardcoded rows after DB is available | MEDIUM | Prefer Supabase query data; only fallback to setup/empty/error states, not fake production data. |
