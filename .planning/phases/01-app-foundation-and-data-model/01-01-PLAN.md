<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Supabase service-role key accidentally exposed to browser bundle | HIGH | Only define publishable/browser-safe keys in `NEXT_PUBLIC_*`; never add service-role keys to client utilities or `.env.example`. |
| Missing environment variables cause ambiguous runtime crashes | MEDIUM | Centralize env validation in `src/lib/env.ts` and render a clear setup state on the dashboard. |
| Student portal/auth scope creeps into Phase 1 | MEDIUM | Keep auth utilities instructor-oriented and avoid student login UI/routes. |
</threat_model>
