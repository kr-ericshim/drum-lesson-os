<threat_model>
## Threat Model

| Threat | Severity | Mitigation |
|--------|----------|------------|
| Detail route leaks another instructor's student by id | HIGH | Use Supabase authenticated/RLS query path only; do not add service-role clients or public bypasses. |
| Missing student id crashes the route | MEDIUM | Return a quiet `Student not found` state with a roster return link. |
| Tabs hide core context behind secondary navigation | MEDIUM | Default summary tab must include the core pre-lesson memory fields. |
</threat_model>
